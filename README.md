## 📚 Learning Path

### Phase 1: Core Fundamentals
- **Variables & Data Types**: Understanding primitive and reference types
- **Control Structures**: Conditionals and loops for program flow
- **Methods**: Function creation and parameter handling
- **Arrays**: Single and multi-dimensional array operations

### Phase 2: Object-Oriented Programming
- **Classes & Objects**: Core OOP principles
- **Inheritance**: Extending classes and method overriding
- **Polymorphism**: Method overloading and runtime polymorphism
- **Encapsulation**: Access modifiers and data hiding
- **Abstraction**: Abstract classes and interfaces

### Phase 3: Advanced Topics
- **Exception Handling**: Try-catch blocks and custom exceptions
- **File I/O**: Reading and writing files
- **Collections Framework**: Lists, Sets, Maps, and Iterators
- **Generics**: Type-safe programming
- **Multithreading**: Concurrent programming basics

## 🚀 Quick Start Guide

### Prerequisites
- Java 17 or later installed on your system
- Git installed for version control
- IntelliJ IDEA CE (recommended) or any Java IDE

### Setup Instructions

1. **Clone the Repository**
   ```bash
   git clone https://github.com/AshC2004/Java_Learning.git
   cd Java_Learning
   ```

2. **Open in IDE**
   - **IntelliJ IDEA**: `File → Open → Select Java_Learning folder`
   - **VS Code**: `File → Open Folder → Select Java_Learning`
   - **Eclipse**: `File → Import → Existing Projects into Workspace`

3. **Run Examples**
   ```bash
   # Compile and run a Java file
   javac src/basics/HelloWorld.java
   java -cp src basics.HelloWorld
   
   # Or use your IDE's run configuration
   ```

4. **Explore the Code**
   - Start with files in the `basics/` directory
   - Each file contains comments explaining concepts
   - Run examples to see output and understand behavior
